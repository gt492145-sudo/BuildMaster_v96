# BuildMaster 企業級升級清單（三類更新）v1

## 已完成（本次）

## 1) 安全關鍵

- iOS Keychain 機敏資料保存強化為 `WhenUnlockedThisDeviceOnly`
- iOS 雲端請求已統一帶入：
  - `X-Client-Nonce`
  - `X-Client-Timestamp`
  - `X-Body-SHA256`
  - `X-DeviceCheck-Token`
- App 端量測資料簽章/驗章與舊資料遷移已啟用
- ATS 額外明確設定：`NSAllowsArbitraryLoads = NO`

## 2) 核心執行路徑

- IBM/OpenAI 相關請求改為 `ephemeral URLSession`
- 針對暫時性網路錯誤與可重試 HTTP 狀態碼加入退避重試：
  - retryable: `408, 425, 429, 500, 502, 503, 504`
  - URLError: timeout / network lost / host unreachable 等
- 雲端錯誤訊息調整為使用者可操作提示（可用性提升）

## 3) 維運工具鏈（CI/CD）

- 新增 GitHub Actions：
  - 檔案：`.github/workflows/ios-enterprise-quality.yml`
  - 內容：
    - iOS 專案自動 build（Simulator）
    - 安全基線檢查（禁止 `NSAllowsArbitraryLoads = YES`）
    - 檢查必要安全標頭是否存在於 Swift 程式
