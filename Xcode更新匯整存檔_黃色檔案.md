# Xcode 更新匯整存檔（黃色檔案）

本檔作為 iOS 原生專案 `LiDARRangefinder` 的集中存檔，彙整本輪已完成且可編譯通過的更新內容。

## 匯整日期

- 2026-03-11

## 本輪已完成更新

1. AR 標靶與追蹤穩定
   - 標靶資源改為最新 `SITE_MARKER_V7`。
   - 追蹤數量調整為一次 1 張圖，提升穩定度。
   - 加入失鎖 debounce 與 overlay 更新節流，降低抖動與閃爍。

2. AR 疊圖與 3D 顯示
   - 保留既有牆面/管線/鋼筋 overlay。
   - 新增高可見紅色 3D 方塊，作為藍圖鎖定 debug 可視化。
   - 新增點擊 AR 畫面可放置 0.5m 混凝土塊，並回寫面積與體積結果。

3. 裂縫 AI 流程
   - 裂縫分析改為「鏡頭即時」優先，不再依賴先選照片。
   - UI 文案更新為即時分析流程。

4. 量子流程
   - 手動啟動可在空口令時使用預設口令啟動。
   - 藍圖首次鎖定可觸發量子最佳化。
   - 新增任務去重冷卻與執行鎖，避免重複送 IBM 任務。
   - 新增 `QuantumManager.swift`，統一 IBM Runtime API 呼叫與錯誤處理。

5. UI 視覺
   - 上下控制面板改為毛玻璃材質，並加入圓角描邊與青藍光暈。
   - 3D 體積掃描頁面改為可拖曳半屏，避免完全遮擋主畫面。
