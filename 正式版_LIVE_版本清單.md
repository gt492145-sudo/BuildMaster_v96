## 本版重點

- 單一 Web 正式版：`index.html` 模組化載入；放樣經 `stake.html` 切換模式後與主程式共用同一套程式
- 價格表資料固定為 118 筆，作為目前正式版基準資料
- 統一正式版視覺識別，降低操作與辨識混淆
- 計算主系統為 `V9.6` 產品線，維持量子更新能力（IBM Quantum API 可選接入、A/B/C + 雷射融合評分、量子一鍵補齊流程）
- 已移除獨立單檔放樣頁（舊 V4.2 HTML），避免與 V9.6 主線重複維護

## 會員等級規劃

正式版採用 3 個會員等級，功能權限如下：

1. 基礎會員：可使用基本功能。
2. 進階會員：可使用基礎 + 進階功能。
3. 最高級會員：可使用全部功能（含所有進階與專業模組）。

## 原生 App 版本規劃（保留）

- iOS 原生版：`LiDARRangefinder`（SwiftUI + ARKit）已建立並納入版本庫
- 目前版本狀態：`v69` 正式納入量子更新能力（量子核心 + 三項任務融合）
- 對外策略：維持 Web 正式版主線，同步保留 iOS 原生量子版本作為高精度場景主力
- 維護原則：持續保持 Web 與 iOS 的 QA/CSV 欄位規格一致，降低後續整合成本
