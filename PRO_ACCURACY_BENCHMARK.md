# Construction Master Pro 精準度驗收基準（9645 → 9646）

> **目的**：在開 IAP 訂閱之前，兩項 Pro 功能（看圖／BIM 自動計算、LiDAR AR）必須達到「現場可信任」的專業準度。  
> **原則**：9645 = 試用 + 精準強化；9646+ = 達標後才接 StoreKit / Play Billing。

---

## 1. 版本對照

| 版本 | Build | 訂閱 IAP | 精準度要求 |
|------|-------|----------|------------|
| 9644 | 審核中 | 無 | AR 開放、五模式、AdMob |
| **9645** | 本地／下一版上傳 | **無** | Pro 試用 + **極強精準預設** + **未審核不可匯出** |
| **9646+** | 達標後 | **有** | 同 9645 門檻，且通過下方 Go 表 |

## 2. 功能 A：看圖／BIM 自動計算

### 2.1 9645 已實作行為（原生 App 殼）

- 「看圖自動判讀」→ 自動改走 **極強精準辨識**（不再走快速模式）
- 信心門檻最低 **95%**（`advAutoInterpretGate`）
- **未按「審核通過／修正後通過」** → 阻擋：
  - 吸入清單（`calculateAndAdd`）
  - 第三頁精算報表匯出
  - BIM 自動計算、施工包／放樣點／QA 匯出

### 2.2 固定驗證圖集（每版回歸）

| # | 圖紙來源 | 預期工項類型 | 參考尺寸（人工量） | 備註 |
|---|----------|--------------|-------------------|------|
| T1 | `test-blueprint-1.png` | 依圖面 | 記錄 v1×v2×v3 | 自動測試腳本用 |
| T2 | `test-blueprint-2.png` | 依圖面 | 同上 | |
| T3 | `test-blueprint-3.png` | 依圖面 | 同上 | |
| T4 | `test-blueprint-4.png` | 依圖面 | 同上 | |
| T5 | `test-blueprint-5.png` | 依圖面 | 同上 | |
| T6 | `test-blueprint-6.png` | 依圖面 | 同上 | |
| T7 | `test-blueprint-7.png` | 依圖面 | 同上 | |
| F1 | 現場常用平面（你手邊） | 牆／柱／版 | 卷尺複核 | 每版至少 3 張 |
| F2 | D棟 C2 梯間相關 | 梯／平台 | 對 `stake-samples/` CSV | 與放樣對位一致 |

**每張圖記錄欄位**（建議用試算表）：

- 工項類型是否正確（Y/N）
- 長×寬×高誤差（cm）
- 數量誤差
- 是否需「修正後通過」
- 極強精準共識分數（%）
- 單次耗時（秒）

### 2.3 Go / No-Go（開 IAP 前必達）

| 指標 | 門檻 | 量測方式 |
|------|------|----------|
| 類型正確率 | **≥ 90%** | 驗證圖集 T1–T7 + F1–F2 加總 |
| 平面尺寸誤差 | **≤ ±3 cm** | 與人工卷尺／CAD 標註比對 |
| 需人工退回比例 | **< 15%** | 「審核退回」次數 ÷ 總跑數 |
| 未審核誤匯出 | **0 次** | 9645 gate 自動阻擋（需手動抽測確認） |
| 連續 20 次判讀 | **0 crash** | 同一裝置 stress test |

**Blocker（任一即 No-Go）**

- 快速模式在原生 App 仍可匯出（應被 gate 擋住）
- 審核 pending 仍可吸入清單
- 尺寸誤差 > ±10 cm 且未標需複核

## 3. 功能 B：LiDAR AR

> 詳細步驟見 `ARPrototype/Docs/封測版_驗收清單_v1.md`、`封測版_現場快速版_v1.md`

### 3.1 必測項目（Pro 相關）

| 項目 | 通過標準 |
|------|----------|
| 標靶／圖紙鎖定 | 10 次中 **≥ 8 次** 3 秒內鎖定（≥ 80%） |
| 圖紙貼牆對位 | 定比例 + 錨點後，overlay 不漂移 |
| 單點量測重複性 | 同點連續 5 次，**標準差 ≤ 2 cm** |
| 設計距離偏差 | 圖面標距 vs AR 實測 **≤ ±3 cm**（`deviationToleranceCm` 預設） |
| CSV 匯出 | 欄位完整、可回灌 Web 量測紀錄 |
| 穩定性 | 連續使用 **20 分鐘 0 crash** |

### 3.2 Go / No-Go（開 IAP 前必達）

| 指標 | 門檻 |
|------|------|
| AR 鎖定成功率 | ≥ **80%** |
| 放樣 QA 分數 | ≥ **85**（對應 `ARPrototype/INTEGRATION.md`） |
| 偏差檢核 | 超過 ±3 cm 時 UI **明確警示**，不可靜默匯出 |
| 主 App 合併度 | `ARPrototype/Native` 高精度栈已併入 Release（非僅橋接 stub） |

**Blocker**

- 啟動 AR crash
- 量測 CSV 缺欄或單位錯誤
- 無 LiDAR 機型未優雅降級（需提示，不可白屏）
