# Construction Master 封測版現場快速清單 v1

用途：現場 10-20 分鐘快速驗收，只保留必測項。  
對應功能：任務 A/B/C/D（AR 藍圖、體積掃描、裂縫抓漏、量子核心）。

---

## 0. 測試前 1 分鐘確認

- [ ] 使用支援 LiDAR 的 iPhone，App 為最新 build
- [ ] 相機/相簿/麥克風/語音辨識權限已允許
- [ ] 標靶圖已備妥（紙本或第二螢幕）
- [ ] 光線正常、避免過暗與強反光

---

## 1. 四大任務必測（主流程）

### A. AR 藍圖透視
- [ ] 3 秒內可鎖定標靶並出現 3D 鋼筋疊圖
- [ ] 移開標靶後可正常失鎖移除
- [ ] 調整主筋/間距/保護層後，畫面可即時更新

### B. 3D 體積掃描（m² + m³）
- [ ] 點擊掃描後有回應，不當機
- [ ] 產出 `m²` 與 `m³`，並顯示取樣點數
- [ ] 同場景連續掃描 3 次，結果沒有異常跳動

### C. AI 裂縫抓漏
- [ ] 可選圖並完成分析，不崩潰
- [ ] 顯示裂縫框線、最長裂縫長度（cm）、嚴重度
- [ ] 無圖分析時有提示（非閃退）

### D. 量子核心戰術模式
- [ ] 手動口令可啟動/解除
- [ ] 語音口令可啟動（有轉寫文字）
- [ ] 啟用後可看到核心等級與戰術建議

---

## 2. 快速回歸（必要）

- [ ] QA 模式切換正常
- [ ] 記錄量測可用（門檻不足時應正確鎖定）
- [ ] CSV 匯出可用
- [ ] AI 助手分析與一鍵套用可用

---

## 3. Go / No-Go 快速判定

### 任何一項命中即 `No-Go`
- [ ] App 啟動或測試中發生崩潰
- [ ] 任一核心任務主流程無法完成（A/B/C/D）
- [ ] 相機或 AR 主流程不可用

### 全部通過可判定 `Go`
- [ ] 四大任務主流程皆可完成
- [ ] 10 分鐘連續操作無崩潰、無卡死
- [ ] 關鍵輸出（m²/m³、裂縫結果、量測記錄）皆可產生
