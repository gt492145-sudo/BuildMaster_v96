# Construction Master 封測版驗收清單 v1

此清單對應目前 `main` 已完成的四大功能：
- 任務 A：AR 藍圖透視
- 任務 B：3D 體積掃描
- 任務 C：AI 裂縫抓漏
- 任務 D：量子核心戰術模式

---

## 0. 測試前準備

- [ ] iPhone 支援 LiDAR，並已開啟開發者模式
- [ ] App 已安裝最新 build（含 commit `c972681`）
- [ ] 相機、相簿、麥克風、語音辨識權限皆允許
- [ ] 測試標靶圖已印出或在第二螢幕顯示
- [ ] 測試環境光線充足，避免強烈反光

---

## 1. 任務 A：AR 藍圖透視（Image Tracking + 3D 鋼筋）

### 1.1 影像錨點鎖定
- [ ] 開啟 App 後，對準標靶圖可在 3 秒內鎖定
- [ ] 系統狀態顯示 `AR POC：已在 ... 上建立 3D 鋼筋錨點`
- [ ] 螢幕移開標靶後，overlay 可正常失鎖移除

### 1.2 參數調整
- [ ] `鋼筋透視參數` 可調主筋數、箍筋間距、保護層
- [ ] 每項變更可即時反映到 3D 鋼筋疊圖
- [ ] 重新開 App 後，參數仍保留（持久化）

### 1.3 對位微調
- [ ] 平移 X/Y 可正常作用
- [ ] 旋轉、縮放可正常作用
- [ ] `重置微調` 可回到初始狀態

**通過標準**
- [ ] 鎖定穩定率 >= 80%（10 次測試中至少 8 次）
- [ ] 鋼筋參數與對位微調無崩潰、無嚴重卡頓

---

## 2. 任務 B：3D 體積掃描儀（m² + m³）

### 2.1 掃描設定
- [ ] 可調整寬度/長度（m）
- [ ] 可調整網格密度（3x3 ~ 11x11）
- [ ] 設定值可保存於下次啟動

### 2.2 掃描結果
- [ ] 點擊 `執行一次掃描` 後有狀態回應
- [ ] 畫面顯示 `掃描面積 m²`
- [ ] 畫面顯示 `估算體積 m³`
- [ ] 畫面顯示取樣點數

### 2.3 基礎合理性驗證
- [ ] 同場景連續掃描 3 次，結果偏差在可接受範圍
- [ ] 改變寬度/長度後，`m²` 與 `m³` 方向性正確（面積變大，體積不應反向下降）

**通過標準**
- [ ] 掃描流程穩定可重複
- [ ] `m²` 與 `m³` 顯示一致且邏輯正確

---

## 3. 任務 C：AI 裂縫抓漏（本地 Vision）

### 3.1 影像流程
- [ ] 可從相簿選圖
- [ ] 點擊 `開始裂縫分析` 可完成本地推理
- [ ] 無圖時按分析有正確提示，不會崩潰

### 3.2 結果呈現
- [ ] 疑似裂縫以框線疊圖顯示
- [ ] 可顯示最長裂縫長度（cm）
- [ ] 可顯示嚴重度（高/中/低）
- [ ] 可調整像素與公分校正比例

### 3.3 例外與誤檢
- [ ] 對非裂縫圖片，系統可提示未找到明顯裂縫
- [ ] 對模糊圖不崩潰，能回傳失敗提示

**通過標準**
- [ ] 分析流程可連續執行 >= 20 次不崩潰
- [ ] 結果框線可視且長度欄位可解讀

## 4. 任務 D：量子核心戰術模式

### 4.1 手動/語音啟動

- [ ] 語音口令可啟動（有轉寫文字）
- [ ] 口令錯誤時有明確提示

### 4.2 聯動策略
- [ ] 啟用後自動開啟最高等級鎖定（Ultra）
- [ ] 啟用後自動開啟自動連續矯正
- [ ] 解除後狀態正確回復

### 4.3 監控與記錄
- [ ] 顯示量子核心等級（%）
- [ ] 顯示戰術建議（自動觸發提示）
- [ ] 戰術記錄有新增、可清空、重開後保留

**通過標準**
- [ ] 手動與語音皆可穩定觸發
- [ ] 聯動狀態無衝突（不會出現已解除但仍顯示啟用）
