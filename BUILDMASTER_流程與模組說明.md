# Construction Master v6.9 流程與模組說明

本文件給現場使用者與維運人員快速了解系統，涵蓋「從登入到輸出」完整流程，並說明各模組用途。

---

## 1) 系統入口與安全模組

### 1.1 保護模式登入
- 進站後先看到「Construction Master 保護模式」。
- 輸入存取碼後才能進入主畫面。
- 目前存取碼：`BuildMaster@2026!`

### 1.2 網域白名單
- 允許網域：
  - `*.github.io`
  - `*.netlify.app`
  - `localhost`
  - `127.0.0.1`
- 非白名單網域會被鎖定並顯示未授權訊息。

### 1.3 安全限制
- 阻擋 iframe 內嵌顯示。
- 前端防護：右鍵、部分開發者快捷鍵限制。
- 畫面右下有安全浮水印（含主機與時間）。

## 2) 主流程（一般使用者）

### 步驟 A：選地區與材料單價
1. 在「地區資料來源」選擇城市或自動定位。
2. 系統載入對應 `prices-*.json`。
3. 若地區資料太少（偏少筆數），會自動改用全台完整價目，避免只看到少量資料。
4. 用「搜尋材料關鍵字」篩選後，按「套用到單價欄」。

### 步驟 B：圖紙量測與試算
1. 上傳圖紙。
2. 先做「定比例」再量測距離。
3. 選工種公式，輸入尺寸/數量/單價。
4. 檢查即時預覽結果，按「加入清單」。

### 步驟 C：BIM 檢核與估價（可選）
1. 上傳 IFC 檔。
2. 看 QA 卡片：總實體、構件數、分數、數量統計。
3. 按「BIM 自動估價預覽」生成估價列。
4. 若有未匹配項目，用「未匹配修復精靈」單筆或批次補規則。
5. 確認後按「匯入估價到清單」。

### 步驟 D：輸出與收尾
1. 主清單可刪除單筆或整體清空。
2. 匯出報表（CSV/Excel流程）。
3. 若誤操作，可用快照回滾。

## 3) 模組一覽（功能定位）

### 3.1 圖紙模組
- 功能：上傳圖面、縮放、定比例、點選量測、清除標註。
- 用途：把圖面尺寸轉為可計算數據。

### 3.2 試算公式模組
- 功能：不同工種公式（混凝土、模板、鋼筋、土方等）。
- 用途：標準化計算數量與金額，降低人工誤差。

### 3.3 材料價目模組
- 功能：讀取 `prices.json` 與各地區 `prices-*.json`，支援搜尋與套價。
- 用途：讓現場估價使用最新價格資料。

### 3.4 BIM IFC 模組
- 功能：解析 IFC 文字、統計 Entity/Type、QA 警示。
- 用途：先做模型品質檢查，再做估價。

### 3.5 BIM 匹配規則模組
- 功能：IFC 類型對應材料（可新增、刪除、匯入、匯出、重置）。
- 用途：提高自動估價準確度，避免錯配。

### 3.6 未匹配修復精靈
- 功能：列出未匹配 IFC 類型，支援單筆/批次套材。
- 用途：快速補規則，減少手工逐筆編輯時間。

### 3.7 單位換算模組
- 功能：長度/面積/體積/重量/常見計數單位換算。
- 用途：避免單位不一致導致估價偏差。

### 3.8 操作日誌模組
- 功能：記錄關鍵操作（匯入、匯出、修復、回滾等）。
- 用途：QA 追溯與責任釐清。

### 3.9 版本快照與回滾模組
- 功能：快照保存與回復（全部、只規則、只估價、只清單）。
- 用途：出錯可快速回復，不怕誤刪誤改。

### 3.10 解說員模組
- 功能：依點擊目標顯示用途與下一步提示。
- 用途：降低新人學習門檻，讓大眾更容易上手。

## 4) 資料儲存位置

### 4.1 本機瀏覽器（localStorage / sessionStorage）
- 主清單資料
- BIM 規則
- 操作日誌
- 快照版本
- 本次分頁登入狀態（session）

### 4.2 專案檔案（Git 管理）
- `index.html`：主程式（UI + JS）
- `prices.json`：全台價目
- `prices-*.json`：各地區價目
- `update_prices_from_arcnet.py`：資料更新腳本
- `.github/workflows/quarterly-price-update.yml`：自動更新流程

## 5) 維運流程（更新版）

### 5.1 開發端更新
1. 修改檔案（例如 `index.html`、價目檔）。
2. 本機測試功能。
3. `git add` / `git commit` / `git push`。

### 5.2 線上同步
- GitHub Pages 會吃 `main` 最新內容。
- 若畫面仍舊版，先做瀏覽器強制重整：`Cmd + Shift + R`。

### 5.3 常見問題排查
- 價目只看到少量：
  - 先看是否已推送成功。
  - 檢查 `prices.json` 是否為 100+ 筆。
  - 確認是否被快取，做強制重整。
- 無法登入：
  - 確認密碼大小寫。
  - 確認網址在白名單。

## 6) 建議標準操作（給一般用戶）

1. 登入保護模式。
2. 選地區並確認「資料筆數」正常。
3. 搜尋材料後套用單價。
4. 進行圖紙量測與工種計算。
5. 有 BIM 就先做 BIM QA，再匯入估價。
6. 每個重要節點先建立快照。
7. 最後再匯出報表與備份規則/快照。

## 8) 會員流程補充（你這版也適用）

### 8.1 目前版本（已上線）
- 目前是「單一存取碼」保護模式，尚未接雲端會員帳號。
- 所有使用者共用同一組存取碼進入（含你說的會員）。
- 同一分頁驗證後可持續使用；關閉分頁或重開瀏覽器需再次輸入。

### 8.2 若要升級為正式會員制（建議下一階段）
- **帳號機制**：每位會員有獨立帳號/密碼。
- **權限分級**：管理員、工程師、檢視者（可設定可見功能）。
- **資料隔離**：會員 A 看不到會員 B 專案。
- **長效登入**：可設定記住登入 7 天/30 天（安全可控）。
- **找回密碼**：Email 驗證或管理員重設。

### 8.3 會員導入後的操作流程（簡版）
1. 會員登入。
2. 選擇或建立專案。
3. 載入價目與 BIM 規則。
4. 量測/估價/快照/匯出。
5. 自動同步到雲端（跨裝置可接續）。
